## Instructions

1. Setup the Flask python environment (one time):

   Windows:

   ```
   py -3 -m venv .venv
   ```

   macOS/Linux:

   ```
   python3 -m venv .venv
   ```

2. Enter the Flask python environment (must repeat each time you reopen your IDE/terminal):

   Windows:

   ```
   .venv\Scripts\activate
   ```

   macOS/Linux:

   ```
   . .venv/bin/activate
   ```

3. Start the server:

   ```
   flask run
   ```

4. Open site in browser:

   http://localhost:5000/
